# 프로젝트 개요

## 한 줄 설명

`parking-radar`는 여행 출발 전에 공항 주차장 만차 여부와 주차 가능 여유를 빠르게 확인하고, 과거 추세와 예상 요금까지 함께 판단할 수 있도록 돕는 공항 주차 대시보드다.

## 문제 정의

- 공항에 출발하기 전에 주차 가능 여부를 확인하기 어렵다.
- 현재 상태만 보면 추세를 알기 어려워, 언제 급격히 혼잡해지는지 판단하기 힘들다.
- 공항마다 주차요금 체계가 달라 대략적인 비용 계산이 번거롭다.

## 목표 사용자

- 자가용으로 공항에 가는 여행객
- 주차 혼잡 패턴을 분석하려는 운영/기획 사용자
- 날짜별, 요일별 주차 패턴을 비교하고 싶은 사용자

## 제품 목표

- 현재 잔여 주차면을 빠르게 보여준다.
- 최근 7일 30분 간격 과거 시계열을 함께 보여준다.
- 시간대별, 요일별 평균 패턴을 시각적으로 보여준다.
- 10대/50대 임계치 하락 및 회복 시점을 분석한다.
- 입력한 시간 기준으로 주차비를 계산한다.

## 범위

포함:

- 김포, 김해, 제주, 인천 등 공항 데이터 조회
- 공항별 주차장별 현재 상태 조회
- 수집 주기 기반 시계열 저장
- 최근 7일 30분 단위 시계열 집계
- 시간대별/요일별 분석
- 임계치 이벤트 탐지
- 주차요금 계산
- 모바일/PC 반응형 웹

제외:

- 로그인/회원 기능
- 개인 맞춤 알림
- 외부 결제 기능
- 인천공항 요금 계산
